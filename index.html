Mobile Carrier Identifier

Identifica el operador móvil de los números telefónicos en Nicaragua.

  • Descargar App
  • Ver en GitHub

Desarrollado y mantenido por altmas5 y chrismeister11

¿Qué es?.

¿Cuantas veces has deseado saber si un número pertenece a la red de X o Y companía telefónica antes de llamar? Esta aplicación para dispositivos con Sistema Operativo Android identifica la compañía telefónica en base a patrones de numeración usados por esta dos empresas.

Si, ya se que existe WhatAmI, pero qué tal si no tengo acceso a internet?

Descargar/Instalar

La aplicación todavía no se encuentra estable pero ya podés descargar la versión alpha. Por ahora es compatible con versiones de android entre 2.1 y 4.2

Cuanto cuesta?

Nada, es gratuita, no incluye publicidad y es de código abierto.
